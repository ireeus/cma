Welcome to Database Help

Registration:

  1. Create an Account:
    • To use the Database, every user needs to register.
    • Go to register and enter required information to set up your username and password.

Login to Sessions:

  1. Login to a Record:
    • Once registered, log in using the login button.
    • You can either select an existing record or create a new one from the dropdown menu.
    • Each record will have a unique date name.
    • If you want to join an existing record, select the existing record from the dropdown menu.
    • To create a new record, select the create new record and enter your details.

Record Access:

  • Each record can be accessed by any registered user.
  • There is 4 types of account (limited, standard, advanced).
    - Limited account is only for viewers and doesn't allow view usernames, to post messages or pictures and to delete any content.
    - Standard account is elevated from limited and allows additionally to view usernames and add or remove own content.
    - Advanced account is elevated from standard and allows additionally to view additional information.
    - Admin account can remove content of all users.
